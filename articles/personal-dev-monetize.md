個人開発を続けながら収益を出したい、というのはエンジニアなら誰もが思うことだ。フリーランス向けの無料ツール集 [AND TOOLS](https://and-tools.net/) を運営してきて、AdSense+アフィリエイトの組み合わせで月10万円という目標をどう設計するかが見えてきた。理論と実体験をまとめる。

## なぜ「無料ツール」サイトがマネタイズに向くのか

ツールサイトの強みは検索意図の明確さにある。「消費税計算」と検索するユーザーは今すぐ計算したい。「税率 計算 サイト」で来るユーザーは目的がはっきりしている。ブログ記事を読みに来るユーザーより商業的価値が高い場合が多い。

もう一つの強みはSEOの観点だ。ツールページはそれ自体が「回答」になる。Googleがゼロクリック検索を増やしている中でも、計算ツールはページを開かないと使えない。ユーザーを確実にページまで連れてこられる。

## 収益の構造を設計する

AdSense+アフィリエイトで月10万円を達成するための逆算を示す。

### AdSense側の計算

```
月間収益 = PV × RPM / 1000
```

ツールサイトのRPMは業種や配置によって大きく変わる。筆者の経験では：

- 一般的なコンテンツサイト: RPM ¥200〜400
- フリーランス向けツールサイト: RPM ¥300〜600
- 金融・税金ジャンル: RPM ¥500〜1,000

月10万円のAdSense収益を目指す場合、RPM ¥500を前提にすると：

```
100,000 ÷ 500 × 1,000 = 200,000 PV/月
```

月20万PVが必要になる。現実的な目標だが、立ち上げ直後には難しい。そこでアフィリエイトを組み合わせる。

### アフィリエイト側の計算

```
月間収益 = PV × CTR × CVR × 単価
```

例として会計ソフトの紹介を入れる場合：

- PV: 100,000
- CTR: 2%（1,000クリック）
- CVR: 3%（30件成約）
- 単価: ¥3,000

= ¥90,000

これで合計収益は AdSense + アフィリエイトで現実的な10万円に近づく。100,000 PVという目標が最初の着地点になる。

## ツールサイトのマネタイズ構成

[AND TOOLS](https://and-tools.net/) では以下の構成を採用している。

### コンテンツ別の収益設計

| ツールカテゴリ | 主な収益源 | 補足 |
|--------------|-----------|------|
| 税金計算系 | AdSense（高RPM）+ 会計ソフト | 金融ジャンルは広告単価が高い |
| ローン計算系 | AdSense + ローン比較 | 金融商品との相性が良い |
| 副業・フリーランス系 | クレカ + 開業届サービス | CVRが出やすいジャンル |
| 時間・単位換算系 | AdSense（低RPM）| 収益より流入源として活用 |

税金・金融系のツールはRPMが高い。フリーランス向けの確定申告ツールや青色申告関連のページには意識して高単価な広告が表示されやすい。ジャンル設計の段階でマネタイズのしやすさを考えることが大事だ。

### ファーストビューにアフィリエイトを仕込まない

ツールサイトでやりがちな失敗が「ツールより前に広告を置く」ことだ。ユーザーはツールを使いに来ている。広告が邪魔をするとすぐ離脱する。

正しい配置の考え方：

1. ファーストビューにツール本体
2. ツールの結果表示エリアの下に関連アフィリエイト
3. ページ末尾に関連ツールへの内部リンク

「計算結果を見た直後」がコンバージョンのゴールデンタイムだ。消費税の計算結果が出たタイミングで「会計ソフトで自動化しませんか？」というCTAを出す。ユーザーの課題感が最高潮のタイミングに合わせる。

## トラフィック獲得の優先順位

マネタイズの前にトラフィックが必要だ。ツールサイトのトラフィック戦略を整理する。

### SEOが最優先

ツールサイトのトラフィックは検索エンジンが主役になる。SNSで拡散するタイプのコンテンツではないためだ。キーワードを選ぶ基準は：

- **検索ボリュームがある**: 月間500回以上を狙う
- **ツールで完結する検索意図**: 「〇〇 計算」「〇〇 換算」
- **競合が弱い**: 大手が参入しにくいニッチを選ぶ

AND TOOLSでは「フリーランス 消費税 計算」「個人事業主 国民健康保険 計算」といったロングテールで記事とツールを組み合わせて流入を増やした。

### ツール数 vs 品質

「とにかくツールを増やす」戦略と「少数精鋭で品質を上げる」戦略がある。筆者の結論は**最初は量で流入源を作り、後から品質を上げる**だ。

理由は単純で、どのツールが検索にヒットするか事前に分からないからだ。AND TOOLSでは34ツールを作った時点で、流入の7割は上位5ツールから来ていた。まずツールを作って、どれが刺さるかを確認する。

## 収益を最大化する3つの施策

### 1. AdSense自動広告を使わない

自動広告は配置をGoogleに任せる仕組みだが、ツールサイトでは逆効果になることが多い。ツールのUI内に広告が差し込まれて使い勝手を壊すケースがある。手動で配置を決める方がRPMの管理がしやすい。

### 2. 内部リンクでPVを伸ばす

ツールサイトの弱点は「1ページで完結してしまう」ことだ。ユーザーが1ツールだけ使って帰ってしまう。これを防ぐために：

- 関連ツールのカード型リンクを結果表示の下に置く
- 「消費税計算」を使ったユーザーに「インボイス番号チェック」を提案する
- 計算ツールの解説記事へのリンクを入れてページ回遊を促す

[ToolShare Lab](https://webatives.com/) では150以上のツールを内部リンクで結んで、セッションあたりのページ閲覧数を増やすことを重視している。なお、ツールサイトで収益を得る場合は特定商取引法に基づく表記が必要になるケースがある。[特商法の書き方ガイド](https://webatives.com/guides/tokusho-writing/) に記載例をまとめているので参考にしてほしい。

### 3. 季節性を使う

税金ジャンルには明確な季節性がある：

- 1〜3月: 確定申告（検索ボリューム最大）
- 6月: 住民税の季節
- 10月〜: インボイス・年末調整

この時期に合わせてコンテンツを更新・追加してトラフィックを伸ばす。確定申告シーズンは平時の3倍以上の流入が来ることもある。事前に準備しておくことで、シーズンインした瞬間に収益化できる。

## ロードマップ：0から月10万円へ

### Phase 1（0〜6ヶ月）: 土台作り

- ツール10〜20本を作る
- AdSenseを貼って実際のRPMを計測する
- どのジャンルのツールが流入を集めるか確認する
- この段階の収益目標: ¥3,000〜10,000/月

### Phase 2（6〜12ヶ月）: スケールアップ

- 流入の多いツールに関連コンテンツを追加する
- アフィリエイトを試行的に導入する
- 月間PVが10,000を超えたらアフィリエイトASPに審査を申し込む
- この段階の収益目標: ¥15,000〜50,000/月

### Phase 3（12ヶ月〜）: 収益最適化

- RPMが低いページの広告配置を見直す
- CVRの高いアフィリエイト案件に集中する
- 新ジャンルへの横展開を検討する
- この段階の収益目標: ¥100,000/月

## 失敗から学んだこと

### 「とりあえずAdSense」では数字が見えない

AdSenseを貼っただけで放置していると「RPMが低いな」で終わってしまう。重要なのは**どのページのRPMが何円か**をページ単位で把握することだ。Googleアナリティクスとアドセンスレポートを結合すると、ページ別の収益が分かる。

筆者の場合、全体のRPM平均は低くても、金融系のツールだけ見ると3〜5倍のRPMが出ていた。このデータがあって初めて「金融系ツールを増やす」という意思決定ができる。

### アフィリエイトは案件選びが9割

アフィリエイトの失敗で多いのは「なんとなくリンクを貼る」だ。ツールサイトでは「このツールを使ったユーザーが次に取る行動」を考えてから案件を選ぶ。

消費税計算ツールを使ったユーザーは「消費税を自動計算できる会計ソフトが欲しい」という需要を持っている可能性が高い。この文脈に合った案件を選ぶと、CVRが上がる。

## まとめ

個人開発でAdSense+アフィリエイトを組み合わせてマネタイズするためのポイントをまとめる：

1. **ジャンル設計**: 高RPMの金融・税金系を意識的に選ぶ
2. **配置設計**: ツール結果表示の直後がコンバージョンのピーク
3. **トラフィック**: SEOファーストでロングテールキーワードを狙う
4. **計測**: ページ単位でRPMとCVRを追う
5. **段階的**: Phase 1で検証、Phase 2でスケール、Phase 3で最適化

月10万円は現実的な目標だが、最短でも1年は見た方がいい。土台を作る期間を焦らずに続けることが最終的に収益に繋がる。
