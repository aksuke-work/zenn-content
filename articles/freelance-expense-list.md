「これ経費にしていいの？」という判断基準を持っていないと、計上できる経費を見逃して余計な税金を払い続けることになる。

[AND TOOLS](https://and-tools.net/) の経費率判定ツールを設計した際に業種別データを調査したが、エンジニアは比較的経費として認められる支出の種類が多い。この記事では、フリーランスエンジニアが経費にできるものを網羅的にリストアップする。

---

## 経費として認められる基本的な条件

税務上の経費として認められるには、**事業に直接関係する支出**である必要がある。判断基準は以下の2点だ。

1. **事業のために使ったか**（業務に関連しているか）
2. **実際に支出したか**（架空の経費は不可）

100%プライベートな支出は経費にならない。逆に事業とプライベートが混在している場合は、合理的な割合で「家事按分」して経費計上できる。

## カテゴリ別 経費リスト

### 1. 機器・設備費（消耗品費 or 工具器具備品）

| 項目 | 科目 | 注意点 |
|------|------|--------|
| PC・MacBook | 消耗品費 or 工具器具備品 | 10万未満は全額即時経費。30万未満は青色申告の少額減価償却特例で全額可 |
| モニター・ディスプレイ | 同上 | 2台目以降も業務用なら可 |
| キーボード・マウス | 消耗品費 | 全額経費 |
| Webカメラ・マイク | 消耗品費 | ミーティング・配信用途 |
| スマートフォン | 通信費 or 消耗品費 | 業務利用割合で按分 |
| タブレット | 消耗品費 or 工具器具備品 | 業務利用なら全額or按分 |
| デスク・椅子 | 工具器具備品 or 消耗品費 | 自宅兼事務所の場合は按分 |
| NAS・外付けHDD | 消耗品費 | バックアップ用途は業務用として認められやすい |

```javascript
// 減価償却の判定ロジック
function getDepreciationMethod(price, isBlueReturn) {
  if (price < 100000) {
    return "全額一括経費（消耗品費）";
  } else if (price < 300000 && isBlueReturn) {
    return "少額減価償却資産特例：全額一括経費（青色申告者のみ）";
  } else {
    return `固定資産として減価償却（耐用年数で分割計上）`;
  }
}

console.log(getDepreciationMethod(80000, true));   // → 全額一括経費
console.log(getDepreciationMethod(250000, true));  // → 少額減価償却資産特例
console.log(getDepreciationMethod(350000, true));  // → 固定資産として減価償却
```

| デザインツール | Figma, Adobe CC | 業務利用分 |

| プロジェクト管理 | Notion, Linear, Jira | 100%経費 |

| ドメイン費 | Xserver, お名前.com | 100%経費 |
| サーバー費 | VPS, クラウド（AWS, GCP） | 100%経費 |
| セキュリティソフト | ウイルス対策ソフト | 業務利用割合 |

### 3. 通信費

| 項目 | 按分の考え方 |
|------|------------|
| インターネット回線（自宅） | 業務利用時間割合または50%按分が一般的 |
| スマートフォン料金 | 業務利用割合（50〜80%が多い） |
| SIMカード（業務専用） | 100%経費 |

按分割合は「合理的な根拠」があれば税務署に説明できる。在宅勤務中心のエンジニアなら70〜80%の業務按分も主張できる。

### 4. 書籍・学習費（新聞図書費 or 研修費）

| 項目 | 科目 | 注意点 |
|------|------|--------|
| 技術書 | 新聞図書費 | 業務関連であれば全額可 |
| Udemy・O'Reilly等 | 研修費 or 図書費 | サブスクリプション型も可 |
| オンラインコース | 研修費 | 業務スキル向上目的 |
| 技術系Youtubeメンバー | 新聞図書費 | 業務関連コンテンツ |
| カンファレンス参加費 | 研修費 | 交通費・宿泊費も含む |
| セミナー受講費 | 研修費 | 業務関連に限る |

---

### 5. 交通費・旅費（旅費交通費）

| 項目 | 注意点 |
|------|--------|
| クライアント先への交通費 | 領収書またはICカード履歴で証明 |
| 開発合宿の交通費・宿泊費 | 事業目的の合理性を説明できる必要あり |
| カンファレンス参加のための移動 | 業務関連であれば全額可 |
| 自家用車（業務使用分） | ガソリン代・駐車場代の業務按分 |

交通費は「いつ・どこへ・何のために」を記録しておく習慣が重要だ。

### 6. 外注費・人件費（外注費）

| 項目 | 注意点 |
|------|--------|
| フリーランスへの業務委託 | 契約書が必要。10万超は支払調書の発行を |
| デザイン・ライティング外注 | 同上 |
| コンサルティング費用 | 業務関連のもの |

外注費は請求書・振込記録・契約書のセットで保管する。

### 7. 広告・マーケティング費（広告宣伝費）

| 項目 | 例 |
|------|----|
| ポートフォリオサイトの維持費 | ドメイン・サーバー |
| 名刺作成費 | 事業用の名刺 |
| SNS広告 | 案件獲得目的の広告出稿 |
| 求人サイトへの出稿 | クラウドワークス等のプレミアム会員 |

---

### 8. 接待交際費（交際費）

| 項目 | 上限・注意点 |
|------|------------|
| クライアントとの会食 | 個人事業主には上限なし（法人は800万まで） |
| ビジネスパートナーとの飲食 | 「誰と・何のために」の記録が必要 |
| お中元・お歳暮 | 取引先向けは可 |

接待交際費は税務調査で確認されやすい項目だ。「誰と・どんな目的で」の記録を残す。

---

### 9. 地代家賃（地代家賃）

自宅兼事務所の場合、家賃の一部を経費にできる。詳細は別記事「自宅兼事務所の家賃按分」に譲るが、面積または時間の割合で按分する。

| 按分方法 | 計算式 | 例 |
|---------|--------|-----|
| 面積按分 | 事業用面積 ÷ 総面積 | 書斎10㎡ ÷ 60㎡ = 16.7% |
| 時間按分 | 業務時間 ÷ 総時間 | 8h ÷ 24h = 33.3% |

---

### 10. 保険料（損害保険料）

| 項目 | 注意点 |
|------|--------|
| 業務用PC・機器の保険 | 機器保険等 |
| PL保険（賠償責任保険） | フリーランス向けの職業賠償責任保険 |
| 火災保険（按分） | 自宅事務所分の按分 |

---

### 11. 消耗品費・雑費

| 項目 | 注意点 |
|------|--------|
| 文房具・ノート | 事業用であれば全額 |
| プリンタートナー・用紙 | 全額 |
| ケーブル・USBハブ等 | 全額 |
| コーヒー・お茶（自宅で業務中） | 認められにくい。クライアント提供のものなら可 |

## 経費にできないもの（よくある誤解）

### NG例

- **趣味の書籍**：業務関連性がないもの（小説、エンタメ漫画等）
- **プライベートの食事代**：友人との飲食は不可
- **スーツ・衣服**：業務用の制服でなければ不可（エンジニアは特に主張しにくい）
- **自宅の電気代全額**：按分なしで全額計上は不可
- **健康食品・サプリ**：業務との関連性の説明が困難

## 経費率の目安

[AND TOOLSの経費率判定ツール](https://and-tools.net/tools/expense-rate/) で確認できるが、フリーランスエンジニアの経費率の目安は次のとおりだ。

```javascript
// フリーランスエンジニアの経費率目安
const expenseRateBenchmarks = {
  "完全リモート（自宅作業中心）": "15〜25%",
  "クライアント先への出張あり": "20〜35%",
  "外注あり（チーム運営）": "30〜50%",
  "設備投資が多い年（開業年等）": "30〜60%",
};
```

経費率が極端に高い（60%以上）と税務調査でチェックされやすくなる。実態に沿った経費計上を徹底することが重要だ。

## まとめ

フリーランスエンジニアが見落としがちな経費をまとめると：

1. **開業費**（開業前の準備費用）
2. **家事按分**（家賃・通信費・電気代）
3. **サブスクリプション費用**（GitHubなど月額ツール）
4. **技術書・オンラインコース**
5. **カンファレンス参加費と交通費**

[AND TOOLS](https://and-tools.net/) では所得税計算・手取り計算など、フリーランスの税金管理に役立つ無料ツールを公開している。経費を正しく計上したあとの税額を確認するのに活用してほしい。

経費の判断に迷ったときは「事業のために使ったか」「合理的に説明できるか」の2点で判断する。確信が持てない場合は税理士に相談するのが確実だ。
