フリーランスになって最初に戸惑うのが「税金の手続きをいつやればいいかわからない」という問題だ。会社員なら年末調整で済んでいたことを、全部自分でスケジューリングする必要がある。

[AND TOOLS](https://and-tools.net/) で税金計算ツールを開発・運用している中で、ユーザーから「税金のタイミングがわからなくて不安」という声を多くもらった。この記事ではフリーランスが1年間でやるべき税務手続きを月別にまとめる。

---

## 全体像：フリーランスに関わる主な税金

まず何の税金があるかを把握する。

| 税金 | 支払いタイミング | 概要 |
|------|----------------|------|
| 所得税 | 3月・6月・11月（予定納税）+ 翌3月 | 1年間の所得に課税 |
| 住民税 | 6月・8月・10月・翌1月の4回 | 前年所得に課税 |
| 個人事業税 | 8月・11月の2回 | 事業所得が290万超の場合 |
| 消費税 | 翌3月（課税事業者のみ） | 売上1000万超または任意登録 |
| 国民年金 | 毎月 | 月額16,980円（2024年度） |
| 国民健康保険 | 月払いまたは年数回 | 前年所得に応じて変動 |

## 月別チェックリスト

### 1月：新年の準備

- [ ] **前年分の帳簿を締める**：12月分の売上・経費を確定させる
- [ ] **領収書・レシートを整理**：カテゴリ別にまとめる
- [ ] **住民税の第4期納付**（1月末まで）

住民税の第4期を忘れる人が多い。カレンダーに必ずリマインダーを入れておく。

```javascript
// 例：税金の支払い期限をリマインダー設定するメモ
const taxDeadlines = {
  "1月末": ["住民税 第4期"],
  "2月16日": ["確定申告 受付開始"],
  "3月15日": ["確定申告 期限", "所得税 納付期限", "消費税 納付期限"],
  "3月末": ["個人事業の開廃業届（前年開業分）"],
  "6月末": ["所得税 第1期予定納税", "住民税 第1期"],
  "8月末": ["個人事業税 第1期", "住民税 第2期"],
  "10月末": ["住民税 第3期"],
  "11月末": ["所得税 第2期予定納税", "個人事業税 第2期"],
};
```

### 2月：確定申告の準備

- [ ] **確定申告書の作成開始**（2月16日〜受付開始）
- [ ] **医療費・寄付金の集計**（控除対象の確認）
- [ ] **青色申告特別控除の確認**（65万控除を取るためにe-Taxが必要）

2月中旬から焦って動くより、1月から帳簿整理を終わらせておくのが正解だ。

### 3月：確定申告・納付

- [ ] **確定申告書の提出**（3月15日まで）
- [ ] **所得税の納付**（3月15日まで）
- [ ] **消費税の確定申告・納付**（3月31日まで、課税事業者のみ）

[AND TOOLSの所得税計算ツール](https://and-tools.net/tools/income-tax/) で事前に税額を試算しておくと、資金計画が立てやすい。

**振替納税を使うと資金繰りが楽になる**：口座振替にすると4月下旬の引き落としになる。現金の確保に余裕ができる。

### 4月：翌年に向けた見直し

- [ ] **今年の節税対策を計画する**（iDeCo、小規模企業共済、経費最適化）
- [ ] **前年の確定申告結果の確認**（還付・追納の確認）
- [ ] **帳簿の運用ルールを整える**

4月は税務署も落ち着いており、税理士への相談がしやすいタイミングだ。

### 5月：国民健康保険の試算

- [ ] **今年の国民健康保険料の試算**
- [ ] **昨年より所得が大きく変わった場合は前納を検討**

国民健康保険料は前年の所得に基づいて6月に確定する。収入が大幅に増えた翌年は保険料が跳ね上がるため、事前に[AND TOOLSの手取り計算ツール](https://and-tools.net/tools/take-home/)で試算しておく。

### 6月：住民税・予定納税

- [ ] **住民税 第1期の納付**（6月末まで）
- [ ] **所得税 予定納税 第1期の納付**（7月31日まで）
- [ ] **国民健康保険料の通知が届く**（6月中旬）

予定納税は前年の税額が15万円以上の場合に発生する。詳細は後述する。

### 7月：予定納税の納付

- [ ] **所得税 第1期予定納税**（7月31日まで）

前年の所得税確定額の3分の1が2回（7月・11月）に分けて徴収される。急に10万円以上の請求が来て慌てるパターンが多いため、資金を確保しておく。

### 8月：個人事業税・住民税

- [ ] **個人事業税 第1期の納付**（8月31日まで）
- [ ] **住民税 第2期の納付**（8月31日まで）

個人事業税の対象：事業所得が290万円（事業主控除）を超える場合。税率は業種によって3〜5%。エンジニア・デザイナーなどは5%が多い。

### 9月：中間見直し

- [ ] **今年の所得を概算する**
- [ ] **節税対策の実施状況を確認**（iDeCoの拠出、小規模企業共済の掛金等）
- [ ] **来年の住民税・国保の試算**

9月時点で今年の収入が確定しつつあるため、このタイミングで節税の打ち手を整理する。[AND TOOLSの経費率判定ツール](https://and-tools.net/tools/expense-rate/)で経費の計上漏れも確認しておく。

### 10月：住民税 第3期

- [ ] **住民税 第3期の納付**（10月31日まで）
- [ ] **年末に向けた経費の整理**

### 11月：予定納税 第2期・個人事業税 第2期

- [ ] **所得税 予定納税 第2期の納付**（11月30日まで）
- [ ] **個人事業税 第2期の納付**（11月30日まで）

11月は支払いが重なる最も資金が出ていく月だ。手元に現金を残しておく必要がある。

### 12月：年度末の締め

- [ ] **今年の収支を確定させる**
- [ ] **購入予定の経費は今年中に支払う**（翌年に繰り越せないものは12月中に）
- [ ] **来年分のiDeCo・小規模企業共済の掛金を確認**
- [ ] **源泉徴収票を取引先から受け取る**

## 資金計画のポイント

### 「手取り率」で資金管理する

フリーランスの請求額から実際に手元に残る金額は、年収によって異なる。

```javascript
// 手取り率の概算計算（独身・東京・青色申告65万控除の場合）
function estimateNetIncomeRate(annualRevenue) {
  // 概算の税率表（所得控除は青色65万+基礎48万で計算）
  if (annualRevenue < 3000000) return 0.75;  // 約75%手元に残る
  if (annualRevenue < 5000000) return 0.70;  // 約70%
  if (annualRevenue < 8000000) return 0.65;  // 約65%
  if (annualRevenue < 12000000) return 0.60; // 約60%
  return 0.55;                               // 約55%
}

// 使い方
const revenue = 6000000; // 年収600万円
const netRate = estimateNetIncomeRate(revenue);
console.log(`手取り概算: ${(revenue * netRate).toLocaleString()}円`);
// → 手取り概算: 3,900,000円
```

正確な数字は [AND TOOLSの手取り計算ツール](https://and-tools.net/tools/take-home/) で試算できる。

### 税金積立の方法

毎月の入金額の25〜30%を税金用口座に積み立てる運用が安全だ。

```
入金100万円の場合：
├── 生活費・事業費口座：70万円
└── 税金積立口座：30万円
    ├── 所得税・住民税用：約15万円
    ├── 国民年金・国保用：約10万円
    └── 予備：約5万円
```

## よくあるミス

### ミス1：予定納税を忘れる
7月と11月に突然届く納付書。前年の所得税が15万円を超えていると自動的に発生する。忘れると延滞税が発生する。

### ミス2：住民税を4回払いと知らない
確定申告後、6月に大きな住民税の通知が届いて驚く人が多い。前年所得に課税されるため、翌年6月まで遅れて課税される仕組みだ。

### ミス3：消費税の免税期間を甘く見る
開業から2年間は原則消費税免税。ただしインボイス登録をした場合はその時点から課税事業者になる。

## まとめ

フリーランスの税金管理で重要なのは「いつ・いくら払うか」を事前に把握することだ。

- **3月**：確定申告・所得税・消費税
- **6〜7月**：住民税第1期・予定納税第1期
- **8月**：個人事業税第1期・住民税第2期
- **10〜11月**：住民税第3期・予定納税第2期・個人事業税第2期
- **翌1月**：住民税第4期

[AND TOOLS](https://and-tools.net/) では所得税・手取り・副業税金など、フリーランスの税金管理に役立つ無料ツールをまとめて提供している。カレンダー管理と並行して、数字の把握に役立ててほしい。
