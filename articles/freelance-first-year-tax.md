開業1年目の確定申告は、誰もが一度は「これで合ってるのか」と不安になる。申告内容の間違いは追徴課税につながり、申告漏れは無申告加算税のリスクがある。

[AND TOOLS](https://and-tools.net/) の税金計算ツール開発を通じて多くのフリーランスの疑問に接してきた経験から、1年目に起きやすい失敗と対策をまとめる。

---

## 開業1年目のスケジュール確認

まず、開業年度に関係する手続きの流れを整理する。

```
[開業時]
├── 個人事業の開廃業届（開業から1ヶ月以内）
├── 青色申告承認申請書（開業から2ヶ月以内 or 3月15日まで）
└── 青色事業専従者給与に関する届出書（専従者がいる場合）

[12月末]
└── 帳簿を締める（白色も青色も必要）

[翌年2月16日〜3月15日]
├── 確定申告書の提出
└── 所得税の納付
```

開業届を出したかどうかにかかわらず、事業所得があれば確定申告は必要だ。

## チェックリスト：申告前に確認すべき20項目

### 【収入の確認】

- [ ] **全取引先からの報酬を集計した**
  - 源泉徴収票・支払調書を全件受け取ったか
  - 振込記録と帳簿が一致しているか

- [ ] **源泉徴収されている報酬を把握している**
  - 源泉徴収済みの税額は確定申告で「精算」される（還付または追納）
  - 源泉徴収税額 = 報酬 × 10.21%（100万超の場合は上記に加算あり）

```javascript
// 源泉徴収税額の計算
function calcWithholdingTax(payment) {
  if (payment <= 1000000) {
    return Math.floor(payment * 0.1021);
  } else {
    // 100万円超の部分は20.42%
    return Math.floor(1000000 * 0.1021 + (payment - 1000000) * 0.2042);
  }
}

console.log(calcWithholdingTax(500000));   // → 51,050円
console.log(calcWithholdingTax(1500000));  // → 204,300円
```

- [ ] **副業・アルバイト収入も含めた**（会社員兼業の場合）

- [ ] **物々交換・現物給付も所得になる**（クライアントからの無償サービス等）

---

### 【経費の確認】

- [ ] **事業用の支出をすべて経費計上した**
  - PC・ソフトウェア・書籍・通信費・交通費
  - 計上漏れは節税チャンスの損失

- [ ] **家事按分できる経費を計上した**
  - 自宅家賃、電気代、通信費の事業按分部分
  - 合理的な計算根拠があれば認められる

- [ ] **10万円以上の資産を減価償却している**
  - 開業初年度に購入したPCが30万円なら、青色申告特例で全額一括計上可能（少額減価償却資産の特例）

- [ ] **開業前の準備費用（開業費）を計上した**
  - 開業前に支払った費用も「開業費」として繰延資産に計上できる
  - 任意のタイミングで償却できるため、黒字の年に多く償却するのが有利

```javascript
// 開業費の均等償却 vs 任意償却
const kaigyohiTotal = 300000; // 開業費合計30万円

// 均等償却（5年）
const annualAmortization = kaigyohiTotal / 5;
console.log(`均等償却: 年${annualAmortization.toLocaleString()}円`);
// → 均等償却: 年60,000円

// 任意償却：黒字の多い年に全額償却できる
console.log("任意償却: 収益が多い年に全額300,000円を経費計上可能");
```

---

### 【控除の確認】

- [ ] **青色申告特別控除を適用している**
  - 65万円控除：e-Tax提出 + 複式簿記
  - 55万円控除：書面提出 + 複式簿記
  - 10万円控除：簡易簿記

- [ ] **基礎控除（48万円）を申告している**

- [ ] **社会保険料控除を計上した**
  - 国民年金：年間約20.4万円（2024年度）
  - 国民健康保険：支払った全額

- [ ] **iDeCoの掛金を小規模企業共済等掛金控除に含めた**

- [ ] **生命保険料控除を申告した**（最大12万円）

- [ ] **医療費控除を確認した**（10万円超 or 所得の5%超の医療費）

- [ ] **ふるさと納税（寄付金控除）を申告した**
  - ワンストップ特例は確定申告を行うと無効になる点に注意

---

### 【申告手続きの確認】

- [ ] **青色申告承認申請書を期限内に提出した**
  - 開業から2ヶ月以内、または当年の3月15日まで
  - 提出していない場合は白色申告（10万円控除のみ）

- [ ] **帳簿が適切に記帳されている**
  - 青色65万控除には複式簿記が必須
  - 会計ソフト（freee・弥生・MFクラウド）を使うのが現実的

- [ ] **e-Taxで提出する準備ができている**
  - マイナンバーカード + カードリーダー、またはID・パスワード方式
  - e-Tax提出で65万円控除（書面は55万円）

- [ ] **振替納税の手続きをした（任意）**
  - 口座振替にすると4月下旬の引き落としに延長される

## よくある1年目の失敗

### 失敗1：青色申告承認申請書を出し忘れた

開業届を出したが青色申告承認申請書を出し忘れたケース。この場合、その年は白色申告（最大10万円控除）しか使えない。翌年の3月15日までに申請すれば翌年から青色申告になる。

65万控除と10万控除の差額は55万円。税率20%の場合で年11万円の差になる。

### 失敗2：源泉徴収を「手取り」だと思い込んでいた

クライアントから「報酬50万円、源泉10万円引いて40万円振り込みます」と言われたとき、売上は50万円として計上する必要がある。40万円を売上にすると過少申告になる。

```javascript
// 正しい帳簿記録
const gross = 500000;  // 売上（50万円）
const withheld = 51050; // 源泉徴収税額（10.21%）
const net = gross - withheld; // 入金額（448,950円）

// 帳簿には
// 売上：500,000円
// 源泉徴収税：51,050円（確定申告で精算）
console.log({ gross, withheld, net });
```

### 失敗3：開業前の費用を諦めた

PCやドメイン代など開業前に使った費用を「経費にできない」と思い込んで諦めるケース。開業費として資産計上し、任意のタイミングで償却できる。

### 失敗4：領収書がない経費を全部諦めた

少額の交通費（Suica利用等）は領収書がなくても「出金伝票」に記録することで経費計上できる。ただし合理性と事業関連性の説明が必要。

## 1年目に整備しておきたい仕組み

### 帳簿ツールの選定

| ツール | 月額 | 特徴 |
|--------|------|------|
| freee | 1,980円〜 | UIがわかりやすい、銀行連携が強い |
| 弥生会計オンライン | 1,100円〜 | 実績が長い、安定感がある |
| マネーフォワードクラウド | 1,298円〜 | 自動仕訳の精度が高い |

会計ソフトを使わない選択肢は避ける。手作業のExcelは青色65万円控除の複式簿記要件を満たすのが難しく、ミスも増える。

### 事業用口座の分離

プライベートの口座と事業用の口座を分ける。経費の按分計算が明確になり、帳簿作成の手間が大幅に減る。

[AND TOOLS](https://and-tools.net/) の各種計算ツールで税額の概算を把握しながら、資金計画を立てると慌てずに申告できる。

## まとめ

1年目の確定申告でやるべきことは「正確な収入の計上」「経費の漏れのない計上」「使える控除を全部使う」の3点だ。

特に**青色申告承認申請書の提出**と**複式簿記での記帳**は、控除額に直結する最重要項目。開業と同時に会計ソフトを導入し、毎月帳簿を更新する習慣をつけることが、申告時の焦りをなくす最善策だ。

税額の試算には [AND TOOLSの所得税計算ツール](https://and-tools.net/tools/income-tax/) が役立つ。実際に申告する前に概算を把握しておくと、資金確保の計画が立てやすくなる。
