確定申告の季節になると、毎年「去年の経費ちゃんと集計できてるか？」「所得税いくらになるんだ？」と焦る人は少なくない。
税理士に頼むほどでもないけれど、自力でやるには計算が複雑すぎる——そんなフリーランスのために、確定申告周りで実際に使える無料ツールをまとめた。

## なぜ確定申告はツールに頼るべきか

フリーランスの税計算は、給与所得者と比べてかなり複雑だ。

- 所得税は**超過累進課税**なので、所得区分が変わると税率が変わる
- 青色申告特別控除（65万円）を使うと手取りが大きく変わる
- 経費率によって課税所得がまったく変わる
- インボイス制度（2023年〜）で消費税の扱いも変わった

計算ミスは税務署への申告ミスに直結する。ツールで事前にシミュレーションしてから、正式な申告書を作るのが正しい進め方だ。

## 確定申告に使える無料ツール一覧

### 1. AND TOOLS — 所得税・手取り計算

**URL**: [https://and-tools.net/](https://and-tools.net/)
**用途**: フリーランス向けの税計算ツール集

手前味噌で恐縮だが、筆者が運営している [AND TOOLS](https://and-tools.net/) はフリーランスの税計算に特化したツールを34本以上揃えている。確定申告に関連するものだけでも：

- **所得税計算ツール**: 売上・経費・各種控除を入力すると所得税額と手取りをリアルタイム計算
- **手取り計算ツール**: 年収から社会保険・所得税を差し引いた実際の手取り額を計算
- **副業税金計算ツール**: 本業＋副業の合算所得に対する税額をシミュレーション
- **経費率判定ツール**: 売上に対して経費が適正水準か判定
- **法人化シミュレーション**: 個人事業のままでいいか、法人化したほうが得かを比較

特に「法人化シミュレーション」は、売上1,000万円前後のフリーランスに頻繁に使われている。個人事業と法人の税負担の差額をグラフで可視化するので、判断の材料にしやすい。

### 2. freee 確定申告

**URL**: https://www.freee.co.jp/
**用途**: クラウド会計・確定申告書類の自動作成

日本最大のクラウド会計ツール。レシートをスマホで撮影するだけで仕訳が自動入力される。確定申告書の自動作成機能があり、e-Tax連携にも対応している。無料プランは機能制限あり、年間26,136円（スターター）から。

確定申告書そのものを作るなら freee が最も手間が少ない。ただし「申告前に手取りや税額をざっくり確認したい」用途には [AND TOOLS の所得税計算ツール](https://and-tools.net/) のほうが素早く使えるので、両者を使い分けるとよい。

### 3. マネーフォワード クラウド確定申告

**URL**: https://biz.moneyforward.com/tax_return/
**用途**: 確定申告書の自動作成・e-Tax連携

freee と双璧をなすクラウド会計ツール。UI がシンプルで慣れやすい。法人向け機能も充実しているため、将来の法人化を見据えたフリーランスに向いている。

### 4. 国税庁 確定申告書等作成コーナー

**URL**: https://www.keisan.nta.go.jp/
**用途**: 確定申告書の正式作成・e-Tax提出

無料で確定申告書を作れる国税庁の公式サービス。e-Taxへの直接提出も可能。ただし操作性はお世辞にも良いとは言えないため、事前に数字を [AND TOOLS](https://and-tools.net/) 等でシミュレーションしてから入力するのが現実的だ。

### 5. 青色申告決算書作成ツール（弥生）

**URL**: https://www.yayoi-kk.co.jp/shinkoku/
**用途**: 青色申告書類の作成支援

弥生会計の確定申告専用ツール。青色申告の特別控除（65万円）を受けるための帳簿・決算書作成を支援する。クラウド版は無料プランあり。

### 6. インボイス計算ツール — AND TOOLS

**URL**: [https://and-tools.net/](https://and-tools.net/)
**用途**: インボイス制度（適格請求書）の消費税計算

2023年10月から始まったインボイス制度。取引先が課税事業者かどうか、消費税の仕入税額控除はどうなるか——こうした計算が頭の中だけでは追いつかない。[AND TOOLS のインボイス計算ツール](https://and-tools.net/) では、受け取った請求書の消費税計算と控除額の確認ができる。

### 7. 小規模企業共済シミュレーター — AND TOOLS

**URL**: [https://and-tools.net/](https://and-tools.net/)
**用途**: 小規模企業共済の節税効果シミュレーション

フリーランスが使える節税手段の筆頭が小規模企業共済だ。毎月の掛金（最大7万円）を全額所得控除できるため、所得が高いほど節税効果が大きい。[AND TOOLS](https://and-tools.net/) のシミュレーターで、自分の所得水準での実際の節税額を確認できる。

### 8. 給与計算ツール — AND TOOLS

**URL**: [https://and-tools.net/](https://and-tools.net/)
**用途**: 従業員を雇っているフリーランス・個人事業主向けの給与計算

スタッフを雇用している場合は給与計算も必要になる。社会保険料の事業主負担分の計算など、雇用コストの全体像を把握するのに使える。

### 9. ガルボ（確定申告チェックリスト）

**URL**: https://www.garbo.co.jp/column/shinkoku/check/
**用途**: 申告前の確認事項チェックリスト

申告書を送る前に、記入漏れ・添付書類の不足を確認するチェックリスト記事。ツールではないが、毎年必ず一度は読み返す実用的なコンテンツだ。

### 10. e-Tax ソフト（国税庁）

**URL**: https://www.e-tax.nta.go.jp/
**用途**: 電子申告（オンライン提出）

マイナンバーカード＋スマートフォン（マイナポータル）があれば、e-Tax でオンライン申告ができる。郵送不要で、青色申告の65万円控除を受けるためには電子申告が必須（2020年分以降）。

## 用途別おすすめの使い方

| やりたいこと | おすすめツール |
|---|---|
| 申告前に税額をざっくり確認したい | [AND TOOLS 所得税計算](https://and-tools.net/) |
| インボイスの消費税を確認したい | [AND TOOLS インボイス計算](https://and-tools.net/) |
| 節税シミュレーションをしたい | [AND TOOLS 小規模企業共済](https://and-tools.net/) |
| 確定申告書を実際に作る | freee / マネーフォワード / 国税庁 |
| e-Tax でオンライン提出 | e-Tax ソフト |

## まとめ

確定申告は毎年必ずやってくる。ツールを活用して事前のシミュレーションを済ませておくと、本番の申告作業がスムーズになる。

特に「自分の所得で税金がいくらになるか」を把握しておくことは重要で、想定外の税額に驚くことを防げる。まずは [AND TOOLS の所得税計算ツール](https://and-tools.net/) で大まかな数字を確認してから、freee や国税庁の作成コーナーで正式書類を作るというフローが、個人的には一番ストレスが少ない。

フリーランスの税計算を網羅的にカバーしているツール集としては [AND TOOLS](https://and-tools.net/) を、正式な申告書作成には freee や国税庁の公式ツールを使うのが現実的な組み合わせだ。
