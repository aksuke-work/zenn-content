フリーランスとして仕事をしていると、見積書・請求書の作成は避けられない業務だ。しかし、毎月の書類作成に時間をかけすぎているのはもったいない。

無料ツールを活用すれば、書類作成の手間を大幅に削減できる。インボイス制度（適格請求書）への対応も含めて、使えるツールをまとめた。

## インボイス制度の基本確認

2023年10月から始まったインボイス制度（適格請求書等保存方式）。フリーランスが課税事業者として登録している場合、クライアントに渡す請求書には以下の記載が必要だ：

- 適格請求書発行事業者の**登録番号**（T + 13桁）
- 税率ごとに区分した**消費税額**
- 税率の記載

既存の請求書テンプレートをそのまま使っている場合、インボイス対応できていないケースがある。各ツールのインボイス対応状況も合わせて紹介する。

## 見積書・請求書作成の無料ツール

### 1. AND TOOLS / ToolShare Lab — 各種計算ツールで金額確認

**URL**: [https://and-tools.net/](https://and-tools.net/) / [https://webatives.com/](https://webatives.com/)
**用途**: 請求金額・税額の事前確認

書類を作る前に「この案件の消費税いくら？」「振込手数料を差し引いたら実際の受取額は？」を確認するのに [AND TOOLS](https://and-tools.net/) が役立つ。インボイス計算ツールでは、税込・税抜の相互変換や、軽減税率・標準税率が混在する場合の計算もできる。

見積書・請求書そのものを作成したい場合は、[無料の見積書作成ツール](https://webatives.com/tools/estimate/) と [請求書作成ツール](https://webatives.com/tools/invoice/) が使える。ブラウザで完結し、登録不要でPDF出力できる。

これらは「数字を正確に把握するツール」として位置付けるとよい。

### 2. freee 請求書

**URL**: https://www.freee.co.jp/invoice/
**用途**: クラウドで請求書・見積書を作成・送付

freee の無料プランで請求書・見積書を作成できる。インボイス制度対応済みで、登録番号や税区分の記載が自動で処理される。

- PDF出力・メール送付が可能
- クライアントごとの管理ができる
- 作成した書類をfreee会計と連携できる（有料プラン）

無料プランは一部機能が制限されるが、書類作成・送付だけなら無料で十分使える。

### 3. マネーフォワード クラウド請求書

**URL**: https://biz.moneyforward.com/invoice/
**用途**: 請求書・見積書・納品書の作成・管理

マネーフォワードの請求書サービス。無料プランでは月10件まで書類作成が可能。インボイス制度対応済みで、登録番号の設定・税率ごとの消費税表示に対応している。

UIがシンプルで、書類の種類（見積書→発注書→納品書→請求書）をそのままステータスで管理できるのが特徴だ。

### 4. Misoca（弥生）

**URL**: https://www.misoca.jp/
**用途**: 見積書・請求書・納品書の作成・電子帳簿保存対応

弥生グループのMisocaは、無料プランで月5枚まで書類作成ができる。インボイス・電子帳簿保存法（2024年〜）の両方に対応しており、法改正への追従が早い。

請求書を郵送代行してくれる機能があり（有料）、クライアントが郵送での書類を求める場合に便利だ。

### 5. Microsoft Excel / Google スプレッドシート テンプレート

**URL（Googleスプレッドシート）**: https://docs.google.com/spreadsheets/
**用途**: カスタマイズ自由なテンプレートで作成

クラウドサービスを使いたくない場合は、Googleスプレッドシートのテンプレートをベースに自作するのも選択肢だ。

「請求書 テンプレート スプレッドシート インボイス」で検索すると、登録番号欄・税率別消費税欄を含んだ無料テンプレートが多数配布されている。カスタマイズして自社（自分）のフォーマットを作れば、継続して使いやすい。

### 6. Invoice Generator（英語）

**URL**: https://invoice-generator.com/
**用途**: 英語圏クライアント向けの請求書作成

海外のクライアントに英語で請求書を送りたい場合に使える。アカウント不要でPDFを即時生成できる。

日本のインボイス制度には非対応だが、海外案件では通貨・税制が異なるため、国内の制度に縛られず使えるシンプルさが利点だ。

### 7. インボイス計算ツール — AND TOOLS

**URL**: [https://and-tools.net/](https://and-tools.net/)
**用途**: 見積・請求金額の消費税計算

[AND TOOLS のインボイス計算ツール](https://and-tools.net/) は、以下のような計算をサポートしている：

- 税抜価格 → 税込価格の変換（10%・8%）
- 税込価格 → 消費税額の逆算
- 複数品目の消費税合計
- 源泉徴収税（10.21%）の計算

特に「税込XX万円でもらった場合、実際の消費税はいくらか」という逆算は、会計処理でよく発生する計算だ。

見積書・請求書に記載する金額を確定する前に、[AND TOOLS](https://and-tools.net/) で数字を確認してからツールに入力するフローにすると、記載ミスを防げる。

## 見積書・請求書の必須記載事項

インボイス対応の請求書には以下が必要だ。テンプレートを選ぶ際に確認しておきたい。

**インボイス対応 請求書の必須項目：**

| 項目 | 内容 |
|---|---|
| 書類名 | 「請求書」と明記 |
| 発行日 | 発行した日付 |
| 発行者名・住所 | フリーランス本人の屋号・氏名・住所 |
| **登録番号** | T + 13桁（インボイス発行事業者のみ） |
| 請求先 | クライアントの会社名・担当者名 |
| 品目・数量・単価 | 各作業の内訳 |
| **税率ごとの金額** | 10%対象・8%対象を分けて記載 |
| **消費税額** | 税率ごとに明記 |
| 合計金額 | 税込の合計 |
| 振込先 | 銀行口座情報 |
| 支払期限 | 末締め翌月末払い等 |

## 見積書と請求書の使い分け

| 書類 | タイミング | 目的 |
|---|---|---|
| 見積書 | 受注前 | 金額の提示・合意 |
| 発注書（受注書） | 受注確認時 | 受発注の確認 |
| 納品書 | 成果物納品時 | 納品の記録 |
| 請求書 | 納品後 | 代金の請求 |

すべての書類を1つのサービスで管理できると、プロジェクトの進捗と請求の流れが一元管理できて便利だ。

## 源泉徴収の扱い

デザイン・Web制作の報酬は、特定の条件下で**源泉徴収の対象**になる場合がある（報酬・料金等の源泉徴収、報酬が100万円を超える部分は20.42%）。

フリーランスとして法人から仕事を受ける場合、請求書に源泉徴収税額を記載するよう求められることがある：

```
報酬金額:     100,000円
消費税 (10%):  10,000円
小計:         110,000円
源泉徴収税額: -10,210円（報酬の10.21%）
請求金額:      99,790円
```

[AND TOOLS の各種計算ツール](https://and-tools.net/) で源泉徴収後の実受取額も事前確認できる。

## まとめ

フリーランスの書類作成でおすすめの組み合わせ：

- **金額・税額の確認**: [AND TOOLS](https://and-tools.net/)（インボイス計算・手取り計算）
- **書類作成・管理（少量）**: freee 請求書 or Misoca（無料プラン）
- **書類作成・管理（多量）**: マネーフォワード クラウド請求書 or freee 有料プラン
- **海外案件**: Invoice Generator
- **カスタムフォーマット**: Googleスプレッドシート + テンプレート

書類作成に時間をかけるより、本業の制作時間を確保することが収益につながる。ツールで自動化できる部分は積極的に任せて、クリエイティブな作業に集中するのが賢い時間の使い方だ。
